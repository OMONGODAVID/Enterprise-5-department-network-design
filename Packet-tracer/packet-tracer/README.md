## Packet Tracer Files

This directory contains the Cisco Packet Tracer implementation of the enterprise
5-department network.

### File Description
- **5-department-network.pkt**
  - Fully functional enterprise-style network
  - 5 departments with separate subnets
  - Wired PCs, wireless clients, VoIP phones, and printers
  - Central router providing DHCP to all devices
  - Inter-department connectivity verified

### Requirements
- Cisco Packet Tracer version 8.0 or later

### Usage
1. Open the `.pkt` file in Cisco Packet Tracer
2. Power on all devices
3. Verify DHCP IP assignment on wired and wireless clients
4. Test connectivity using ping and VoIP calls
