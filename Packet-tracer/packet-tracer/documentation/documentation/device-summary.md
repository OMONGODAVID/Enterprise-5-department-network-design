# Device Summary

This project includes the following device types per department:

## End Devices
- 4 wired PCs
- Wireless laptop
- wireless tablet
- VoIP IP phones
- Network printers

## Network Devices
- 1 Access switch per department
- 1 Wireless Access Point per department
- 1 Central router

## Servers
- Shared server connected to the router
- Provides network services such as DNS and application hosting

## Wireless Integration
- Access Points operate in bridge mode
- Wireless clients receive IP addresses from router DHCP
- Wireless and wired devices communicate seamlessly
